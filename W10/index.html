Validating Forms

I'm still learning how to use Svelte, so I thought this would be a good oportunity to learn a method of form validation with svelte! I read an article that recommended I pair my svelte form with a library caled 'yup', which has turned out to be a pleasure to use! The basic gist of yup is that you define a form schema, and then compare your form data to the schema to validate it. Yup will then give you back all the errors it spots in your form. I went ahead and built a little demo using Svelte and Yup to validate a simple form!

If the preview doesn't start automatically, run ctrl+c and npm run preview

Using Fetch

Fetch is awesome, it's super convienient for sending off quick GET requests with no hassle. I think I still prefer to use libraries like 'axios' because of the more convienient interface axios.post(url, data) for example. With fetch you have to pass in a settings object to configure your HTTP method, which is somewhat annoying. Great for simple GET requests though!
