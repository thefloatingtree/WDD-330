W2 Notes

Reading: "JavaScript: Novice to Ninja, 2nd Edition"

  • Programming Basics
    • "Direct Assignment and Assignment By Reference" is a really interesting facet of JavaScript! It's something that can really bite you in the butt if you don't keep it in mind. Immutable patterns really help in this regard, if you never mutate your objects you pretty much don't have to worry about assignment by reference.
    • Wrapper objects - I actually didn't know that JavaScript creates wrapper objects around primitive data types. I just assumed they were objects from the get go!
    • I didn't know Symbols existed, I'll have to look into them more!
    • Hard equality is good!
  • Arrays, Logic, and Loops
    • I love destructuring! It's so convenient.
    • Sets are also pretty useful! Occasionally I find problems that are simplified a lot by Sets. Also, very useful for removing duplicates in an array :) Also didn't know about the memory leak stuff, I'll defintely keep that in mind going forward!
    • Maps are even more useful than Sets!
    • I don't use for loops so much anymore! I find the ES6 array methods so much more convenient.
  • Functions
    • Love the arrow syntax. Makes creating anonymous functions a breeze!
    • Love map, reduce, and filter. I use them all the time! So much so I hardly ever use a for loop anymore.
